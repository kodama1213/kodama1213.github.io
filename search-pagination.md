# Búsqueda & Paginación

Utiliza la barra de búsqueda (arriba a la izquierda) para encontrar cualquier palabra en la wiki. Abajo verás los botones Anterior/Siguiente para navegar entre páginas.

```js
window.$docsify = {
  search: 'auto',
  pagination: {
    previousText: 'Anterior',
    nextText: 'Siguiente'
  }
}
```
