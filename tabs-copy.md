# Tabs y Copy‑Code

Organiza bloques de código en pestañas (tabs):

```markdown
<tabs>
<tab title="Python">
```python
def hola():
    print('Hola')
```
</tab>
<tab title="JS">
```js
console.log('Hola');
```
</tab>
</tabs>
```
